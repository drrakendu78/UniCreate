FAQ

Winget-focused
Which InstallerType should I choose?

Match your real installer package: EXE for custom installers, MSI for Windows Installer bundles.

Where do silent switches come from?

Use vendor documentation. UniCreate can prefill safe defaults for EXE, but publisher switches stay authoritative.

Why can a GitHub URL fail in review?

Winget validation rejects dead or redirected assets. Keep release assets public and ensure direct download URLs are stable.

When should I publish MSI, EXE, or portable?

Publish what users actually install. If you ship multiple channels, include each as a dedicated installer entry.

Can I edit YAML before submission?

Yes. Review keeps generated manifests editable so you can tune fields before export or PR submission.
