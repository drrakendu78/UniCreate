Core flow

  • Installer URL and hash flow with automatic defaults.
  • Metadata forms for default locale and optional locales.
  • Review step with editable generated YAML tabs.
  • Submit step integrated with GitHub auth and PR creation.

Live operations

  • Recover your recent UniCreate-created PRs from winget-pkgs.
  • Track PR state with clearer badges (Ready, Pending review, Behind, Needs action).
  • Build artifacts and checksums included in release process.
  • In-app update popup with silent Windows update flow.

Workflow

4-step process
  1. 1

    Configure installer links, architecture, and SHA256.

  2. 2

    Fill package metadata and locale details with guided validation.

  3. 3

    Review and adjust generated YAML before exporting or submitting.

  4. 4

    Submit PR to winget-pkgs and monitor live statuses from Home.
